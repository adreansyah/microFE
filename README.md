# microFE

How to running this MFE app.

```sh

cd {YOUR_REPO_DIR}

cd container/ npm install then npm start 
result : localhost:8080

cd header/ npm install then npm start
result : localhost:8081   

cd content/ npm install then npm start
result : localhost:8082

cd footer/ npm install then npm start
result : localhost:8083

cd cart/ npm install then npm start
result : localhost:8084
```
