# microFE

How to running this MFE app.

```sh

cd {YOUR_REPO_DIR}

cd container/  npm start 
result : localhost:8080

cd header/  npm start
result : localhost:8081   

cd content/  npm start
result : localhost:8082

cd footer/  npm start
result : localhost:8083

cd cart/  npm start
result : localhost:8084
```
